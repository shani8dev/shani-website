Features & Architecture

Shanios is engineered to deliver an immutable, secure, and high-performance OS experience. Its innovative design blends a rock-solid, read-only core with a dynamic blue/green update system, advanced container integration, and an intuitive user environment‚Äîideal for OEM deployments and everyday use.

Core Architecture & Blue/Green Updates

Dual-Root Architecture

Two complete system roots for atomic upgrades and immediate rollback.

Shanios maintains two independent roots on a modern copy-on-write filesystem. While one root is active, the other is updated in the background‚Äîensuring that every upgrade is fully validated before switching. If any issues occur, the system can instantly revert.

Automated maintenance routines keep the filesystem balanced and performing at its best.

Immutable Base & Dynamic Overlays

A secure, unmodifiable core with flexible overlays for configuration.

The core system is built as a read-only image to protect critical components from accidental changes. Dynamic overlays manage configuration and volatile data, preserving system integrity while allowing necessary customizations.

Optional full disk encryption during installation further secures your data.

Blue/Green Updates (shani-deploy)

Automated, zero-downtime updates with seamless rollback.

Shanios uses a robust blue/green update strategy where the inactive root is updated in the background using the shani-deploy command. Once the new version is validated, a seamless switch occurs. If any issues arise, an instant rollback ensures continuous stability.

Installation, OEM & Desktop Environment

Modern Live ISO & os-installer

Fast, efficient installation with automatic swapfile and optional encryption.

The live ISO boots into an installer powered by a lightweight tiling window manager for a swift, resource-friendly setup. With integrated os-installer provides a complete testing ground across various hardware platforms.

During installation, a swapfile for hibernation is automatically created (dynamically sized upto ram size) to support seamless session resumption. You also have the option to enable full disk encryption for enhanced security.

OEM Ready GNOME Desktop

A sleek, customizable desktop designed for premium OEM deployment.

After installation, Shanios launches with a minimal GNOME desktop that is both clean and highly responsive. The intuitive initial setup enables effortless personalization, making it perfect for OEM partners and individual users.

A polished boot splash and optimized resource usage deliver a modern, premium desktop experience.

Application Delivery & Container Integration

Flatpak Integration

Secure, sandboxed applications with seamless updates.

Shanios delivers applications as Flatpak packages, ensuring they run in isolated environments that are separate from the immutable core. This simplifies the update process and protects the system from potential software conflicts.

Intuitive Container Management (Pods & BoxBuddy)

Visual tools streamline container orchestration for both OCI containers and distro management.

Shanios integrates advanced graphical interfaces to simplify container management. Pods offers an intuitive GUI for managing OCI containers, while BoxBuddy provides a seamless way to create, upgrade, and integrate containerized Linux distributions alongside your host system.

These tools eliminate the need for complex command-line operations, making container orchestration accessible to all users.

Performance, Security & Productivity

Advanced Resource Optimization

Intelligent process and resource management for smooth performance.

Shanios continuously monitors and adjusts process priorities, balancing tasks across CPU cores to minimize latency and maximize responsiveness. Efficient management of memory and disk operations‚Äîincluding dynamic swap space‚Äîensures peak performance even under heavy workloads.

Enhanced NVIDIA GPU Support

Preconfigured support for NVIDIA GPUs for superior graphics performance.

Shanios is designed to work seamlessly with NVIDIA hardware, providing out-of-the-box driver support that ensures smooth, reliable graphics performance. This built-in capability delivers an optimal experience for gamers and professionals alike.

Enhanced Terminal & Shell Experience

A powerful, customizable command-line environment for boosted productivity.

Shanios offers a refined terminal interface with advanced auto-completion, real-time syntax highlighting, and intuitive history search. Customizable prompts and efficient navigation tools streamline your workflow, making daily command-line operations faster and more enjoyable.

Installation

This page will guide you through the Shanios installation process and provide recommendations.

Requirements

  • At least 64 GB of storage
  • 4 GB of RAM
  • A 64-bit processor
  • USB stick with at least 8 GB capacity
  • Graphics card supporting at least OpenGL 3.3

We set these requirements to ensure a smooth experience and enough space for Btrfs snapshots. If your system meets these, you're good to go ü™ê!

Live ISO Credentials

After you boot into the Shanios live environment, if required use:

  • Username: shani
  • Password: shani

Steps to Install

  1. Boot your PC and open your firmware setup (UEFI/BIOS). Disable Fastboot & Secure Boot if possible.
  2. Ensure SATA is set to AHCI mode. If your device supports UEFI, configure UEFI only.
  3. Create a bootable USB using DD, Etcher, or Rufus. Then boot from it.
  4. Start the installer in the live environment. Follow the guided procedure.

Shanios uses a Sway-powered live ISO, offering a modern and minimal environment for installation. After installation, you'll have a minimal GNOME desktop for daily use.

Updates & Flatpak ü™ê

For updating your system, Shanios provides the shani-deploy command. This handles atomic, dual-root upgrades, ensuring you can roll back instantly if needed. Flatpak ü™ê is used for user software, keeping apps containerized and simplifying your update experience.

Recommendations

  • We do not support Dual booting. Another OS might overwrite your bootloader.
  • We recommend real hardware for the best performance. Virtual machines may lack graphics acceleration.
  • Use shani-deploy for reliability. Running raw pacman is not recommeneded.

About the Project

Shani OS was created with a bold vision: to redefine system updates through an immutable architecture. By ensuring each deployment is validated, secure, and tamper-proof, Shani OS eliminates the risks of broken upgrades. Our atomic update mechanism, combined with a blue-green deployment model, guarantees a seamless and reliable experience‚Äîso you never have to worry about a failed update again.

Shani OS emerged as an independent open-source project developed by Shrinivas Vishnu Kumbhar as the cornerstone of my M.Tech research at Walchand College of Engineering, Sangli, Maharashtra, India beginning in 2023. Building upon my expertise as co-founder of Garuda Linux and professional experience as a DevOps Engineer, I've crafted Shani OS as a groundbreaking truly immutable operating system that redefines digital security.

Our pioneering first release launched on March 8, 2025, marking a significant milestone in operating system design. Shani OS delivers exceptional security and reliability while maintaining an intuitive user experience that anyone can appreciate. We're now expanding into a vibrant community-driven ecosystem, welcoming innovators worldwide to help shape the future of computing through Shani OS.
